Welcome to Guess Number

This is an oversized documentation for my Python program to play "guess a number". The main point of this documentation is to practice the use of Mkdocs.

How to play

You will be prompted to guess a number. Just try it. If you don't guess it correctly the program will tell you if your number was too low or too high. With that information you can try and guess again until you guess the correct number.
